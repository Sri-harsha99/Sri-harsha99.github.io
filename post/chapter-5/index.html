Price Tracking with notification

August 11, 2020

Well, this is August 2020, Six months into the COVID-19 lockdown and I want to play rocket league, so need a controller.

I have been able to scrape the price of the products I need and send notifications If the product price drops below a certain point we chose. This works with both Amazon India and Flipkart. There are many services online which offered the same but they are unreliable.

Now coming to the code, I have used beautiful soup for scraping and use the package named win10 toast For Windows 10 notifications. Code can be easily customized based on our needs and we simply have to change the below lines.

Products is an array where we insert the URLs of the products we want to track. There may be different models and colors for the same product. Insert all such URLs in this array. If the price of any one of the above products goes below the accepted price we get notified. With the to_refresh variable, we can control how often the program checks the product price.
